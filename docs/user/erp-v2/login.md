1. Akses link https://erp.etos.app/auth/login
2. Masukkan `Email` dan `Password`

  ![Login](./img/login.png)
  
3. Klik `Login`

:::note
Bagi user yang belum memiliki akun, silahkan koordinasi dengan SPV untuk minta didaftarkan ke bagian HR (Human Resources)
:::
